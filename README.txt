
IVR Call Manager - Installation Guide

1. Extract the contents of this zip file to any folder on your computer
2. Run IVRCallManager.exe
3. First time setup:
   - Go to the SYSTEM CONFIGURATION tab
   - Enter your Twilio credentials:
     * Account SID
     * Auth Token
     * Twilio Phone Number
   - Click SAVE CONFIGURATION
4. To make calls:
   - Go to the INITIATE CALL tab
   - Enter the target phone number
   - Click INITIATE CALL SEQUENCE

Note: You need a valid Twilio account to use this application.
